Sobre a proposta

Feito para demonstrar os fundamentos apresentados em sala de aula

A ideia é servir como uma base curta e limpa para a apresentação: o front é puro HTML/CSS/JS, e o back usa Node.js, Express, MongoDB, Mongoose e JWT.

Tecnologias utilizadas neste projeto

  • HTML semântico
  • CSS responsivo
  • DOM e eventos
  • Fetch API
  • Express
  • MongoDB
  • Mongoose
  • JWT
